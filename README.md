# 🐾 Pet API (ระบบฐานข้อมูลสัตว์เลี้ยง)

API สำหรับดึงข้อมูลสัตว์เลี้ยงยอดนิยม รวบรวมข้อมูลสายพันธุ์ โภชนาการ การดูแล และโรคประจำพันธุ์ที่ควรระวัง พัฒนาด้วย **Node.js** และ **Express.js**

## 🌟 ฟีเจอร์หลัก (Features)
- รองรับข้อมูลสัตว์ 2 หมวดหมู่: **สุนัข (Dogs)** และ **สัตว์เลื้อยคลาน (Reptiles)**
- ดึงข้อมูลสัตว์เลี้ยงทั้งหมด หรือแยกตามหมวดหมู่ได้
- ค้นหาสัตว์เลี้ยงด้วยคีย์เวิร์ด (รองรับทั้งภาษาไทยและภาษาอังกฤษ)
- มีข้อมูลเชิงลึก เช่น โภชนาการตามช่วงวัย, โรคประจำพันธุ์ และคำแนะนำสำหรับมือใหม่
